## Python開発環境の設定

**※貸与PCにPythonの実行環境が構築できていることを前提とする**

### 1. GitHubからソースコードをダウンロード・解凍

二つの方法でサンプルコードをダウンロードすることができる

- `zip`ファイルをダウンロードする
  `PythonStudy2025-main.zip`というファイル名がダウンロードされる。
  このファイルをPCの**デスクトップ**上に解凍し、フォルダ名を`PythonStudy2025`に変更してください
- `Git`でリポジトリをクローンする
  `GitHub Desktop`等がインストールされた場合は、PCの**デスクトップ**にクローンしてください

### 2. 必要なライブラリ等をインストール

```shell
# Pythonのバージョンを確認する
C:\Users\User>python -V
Python 3.13.3

# Pipのバージョンを確認する
C:\Users\User>pip -V
pip 25.0.1 from C:\Python313\Lib\site-packages\pip (python 3.13)

# ダウンロードしたソースコードのフォルダに移動（ここは、デスクトップに解凍したとする）
cd Desktop\PythonStudy2025
C:\Users\User\Desktop\PythonStudy2025>

# pipコマンドで、必要なパッケージやライブラリをインストールする
# 以下は「C:\Users\User\Desktop\PythonStudy2025>」部分を省略する

# データ解析や機械学習関連
pip install pandas scikit-learn

# 図形出力、ブラウザ上でプログラム実行
pip install matplotlib jupyter

# 形態素解析・然言語処理
pip install mecab-python3 unidic-lite

# 生成AI関連
pip install ollama  
```

### 3. `Jupyter notebook`を起動

```shell
C:\Users\User\Desktop\PythonStudy2025> juoyter notebook
```

ブラウザーが自動的に起動し、`Jupyter`ホーム(`Home`)画面が表示される。

![](img/JupyterNotebook01.png)

### 4. Visual Studio Codeでプログラミング

※自分でモジュールやパッケージを開発する際に、`VSCode`を使うと便利。

1. `VSCode`を起動し、`VSCode`上で以下のフォルダを開く
   `C:\Users\User\Desktop\PythonStudy2025`
2. `VSCode`にPython関連の拡張機能を追加する
   拡張機能「`Python`」で検索し、それをインストールする
3. `VSCode`でプログラムを編集する
4. プログラムを実行する。二つの方法でプログラムを実行できる
   a. `VSCode`画面で実行ボタン（▷）を押すと実行する
   b. コマンドラインで「`python main1.py`」のようにプログラムを実行することもできる
